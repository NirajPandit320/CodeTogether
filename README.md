# 🚀 CodeTogether – Real-Time Developer Collaboration Platform

CodeTogether is a full-stack MERN application that enables developers to collaborate in real time.
It combines live code editing, chat, and room-based collaboration into a single platform.

---

## 🧠 Features

* 🔐 User Authentication (JWT-based)
* 👥 Room-based collaboration
* 💬 Real-time chat (Socket.IO)
* 💻 Live code editor (Monaco Editor)
* ⚡ Real-time code synchronization
* 🟢 Active users in room
* 📦 Scalable backend architecture (MVC pattern)

---

## 🛠️ Tech Stack

### Frontend

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```
git clone https://github.com/your-username/devcollab.git
cd devcollab
```

---

### 2️⃣ Setup Backend

```
cd backend
npm install
```

Create a `.env` file:

```
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
PORT=5000
```

Run backend:

```
node server.js
```

---

### 3️⃣ Setup Frontend

```
cd frontend
npm install
npm start
```

## 🌐 API Endpoints

### Auth Routes

* `POST /api/auth/register` → Register user
